# Business Review Automation System

Python automation integrated with N8N workflows for intelligent review request management.

## Project Overview
Automated system for DFW Mobile Fingerprinting that sends personalized review requests via email after service completion, with human approval through Telegram integration.

## Workflow
1. N8N monitors Square appointments 
2. 30-60 minutes post-appointment → Telegram notification
3. Manual approval → Python script executes
4. Personalized email sent with Google review link

## Technologies Used
- Python 3.x
- N8N workflow automation  
- Square API (customer data)
- Telegram Bot API
- SMTP email delivery
- privateemail.com email service

## Features
- Quality control through human approval
- Integration with existing business workflows
- Personalized customer communication
- Direct Google My Business review links

## Business Impact
- Streamlined review collection process
- Improved local SEO through consistent reviews
- Maintains service quality standards
